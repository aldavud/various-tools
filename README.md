- show-inspector.py

Output example: 
> $ python show-inspector.py
>

>
>   CPU: physical: 2 / logical: 16
>
>   RAM: 12.0 GB
>
>   Disks:
>
>     - /dev/sda: 500
>
>     - /dev/sdb: 500
